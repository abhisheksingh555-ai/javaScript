# JavaScript vs ECMAScript

JavaScript:
- A programming language used to build interactive web applications.
- Implemented in web browsers.
- Follows ECMAScript standards.

ECMAScript:
- A specification or standard.
- Defines how scripting languages should work.
- Created by Ecma International.

Example Analogy:

ECMAScript = Rules / Standard  
JavaScript = Language that follows those rules

---

# Features of JavaScript

- Lightweight language
- Interpreted language
- Object-oriented
- Event-driven
- Cross-platform
- Runs on both client and server
